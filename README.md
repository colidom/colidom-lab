## 🛠️ Herramientas Incluidas

### 💻 Dev Tools

#### 🔵 Formateador JSON
Formatea y valida código JSON con resaltado de sintaxis
- ✓ Validación en tiempo real
- ✓ Indentación personalizable
- ✓ Detección de errores
- ✓ Copia con un clic

#### 🟠 Formateador XML
Organiza y valida documentos XML
- ✓ Validación de sintaxis
- ✓ Indentación automática
- ✓ Limpieza de código

#### 🟣 Base64 Converter
Codifica y decodifica texto en Base64
- ✓ Codificación/decodificación bidireccional
- ✓ Interfaz simple y rápida
- ✓ Soporte para grandes textos

#### 🟡 Hex Converter
Convierte texto a hexadecimal y viceversa
- ✓ Conversión instantánea
- ✓ Útil para depuración
- ✓ Formato legible

#### 🔴 Paleta de Colores
Genera paletas armónicas desde un color base
- ✓ Monocromáticas, análogas, triádicas
- ✓ Modelo HSL
- ✓ Códigos HEX exportables
- ✓ Selector de color interactivo

#### 🌹 Regex Tester
Prueba expresiones regulares en tiempo real
- ✓ Resaltado de coincidencias
- ✓ Patrones comunes predefinidos
- ✓ Soporte completo de flags
- ✓ Detalles de grupos de captura

#### 🔵 Timestamp Converter
Convierte entre fechas y timestamps Unix
- ✓ Bidireccional (fecha ↔ timestamp)
- ✓ Múltiples formatos (ISO, UTC, Local)
- ✓ Tiempo actual en vivo
- ✓ Soporte segundos y milisegundos

### 🔧 Utilidades

#### 🟢 Generador QR
Crea códigos QR personalizados
- ✓ URLs, texto, WiFi, vCard
- ✓ Logos personalizados
- ✓ Exportación PNG/SVG
- ✓ Alta calidad

Convierte direcciones a coordenadas GPS

- ✓ Múltiples algoritmos simultáneos

## 🚀 Cómo Empezar

### Prerrequisitos
- Node.js 16+ 
- npm o yarn

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/colidom/colidom-lab.git

# Navegar al directorio
cd colidom-lab

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm start
```

La aplicación se abrirá en `http://localhost:3000`

### Construir para Producción

```bash
npm run build
```

## 🤝 Contribuciones

Las contribuciones son bienvenidas. Si tienes una idea para una nueva herramienta o mejora:

1. Haz un fork del repositorio
2. Crea una rama para tu feature (`git checkout -b feature/nueva-herramienta`)
3. Commit tus cambios (`git commit -m 'feat: Añadir nueva herramienta'`)
4. Push a la rama (`git push origin feature/nueva-herramienta`)
5. Abre un Pull Request

### Guías de Contribución
- Sigue las convenciones de código del proyecto
- Añade tests si es posible
